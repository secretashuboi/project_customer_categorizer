# 📊 Customer Categorizer using Machine Learning

A Machine Learning web application that predicts the customer category based on demographic and purchasing behavior. The model is built using **Logistic Regression** and deployed with **Streamlit**.

## 📌 Project Overview

This project predicts customer categories using machine learning. It allows users to enter customer details through an interactive Streamlit interface and instantly receive a predicted customer category.

This project demonstrates the complete machine learning workflow:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training
* Model Evaluation
* Model Deployment with Streamlit

## 🤖 Machine Learning Model

* **Algorithm:** Logistic Regression
* **Problem Type:** Multi-Class Classification
* **Output Classes:** 0, 1, 2

---

## 📊 Input Features

The model uses the following customer information:

* Age
* Education
* Marital Status
* Parental Status
* Children
* Income
* Total Spending
* Days as Customer
* Recency
* Wines
* Fruits
* Meat
* Fish
* Sweets
* Gold
* Web Purchases
* Catalog Purchases
* Store Purchases
* Discount Purchases
* Total Promo
* NumWebVisitsMonth

## ▶️ Run the Project Locally

Clone the repository:

```bash
git clone https://github.com/secretashuboi/project_customer_categorizer.git
```

Move into the project directory:

```bash
cd project_customer_categorizer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
